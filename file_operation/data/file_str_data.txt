保护继承是介于公有继承和私有继承之间的一种继承方式。
它在派生类中派生出的成员变量和成员函 数都是保护的，
不能被派生类的对象直接访问，但是可以在派 生类的内部和派生类的子类中进行访问。
保护继承方式体现了“derived-from”的关系， 即派生类是基类的派生和扩展，
但不希望外部使用派生类中的成员。
